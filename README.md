## Descripción

Portafolio profesional donde muestro mi perfil como desarrollador, proyectos personales, habilidades técnicas y un formulario de contacto. Construido como static site y desplegado en **GitHub Pages** con dominio personalizado.

## Tecnologías utilizadas

- **HTML5** — estructura semántica y meta tags (Open Graph, Twitter Card)
- **CSS3** — diseño responsive, variables CSS, glassmorphism, animaciones con Intersection Observer
- **JavaScript** — navegación mobile (hamburger menu) y animaciones al hacer scroll
- **GitHub Pages** — hosting y despliegue continuo
- **FormSubmit** — backend del formulario de contacto (sin servidor propio, 100% gratuito)

## Ejecutar localmente

```bash
# Clonar el repositorio
git clone https://github.com/eldevazteca/eldevazteca.github.io.git

# Entrar al directorio
cd eldevazteca.github.io

# Abrir en el navegador (o usa Live Server si lo prefieres)
open index.html
```

Al ser un sitio estático puro, no necesita dependencias ni build step.

## Personalización

- **Avatar**: reemplaza `assets/myicon.png`
- **Redes sociales**: actualiza los enlaces de GitHub y LinkedIn en `index.html` con tus URLs reales
- **Proyectos**: agrega más `.project` cards en `index.html` con las tecnologías que uses
